PDF 압축 관련 FAQ

PDF 압축은 어떤 방식으로 작동하나요?

PDF의 각 페이지를 canvas에 렌더링한 뒤 JPEG 이미지로 변환하고, 그 이미지를 새 PDF에 삽입하는 방식으로 작동합니다.

압축 후 텍스트 선택이 가능한가요?

대부분의 경우 압축 결과물은 이미지 PDF가 되므로 텍스트 선택이나 검색이 어려울 수 있습니다.

어떤 PDF에서 압축 효과가 좋은가요?

스캔본, 사진이 많은 문서, 이미지가 포함된 과제 제출용 PDF에서 효과가 좋습니다. 텍스트 위주의 PDF는 압축 효과가 작을 수 있습니다.

PDF 분할 관련 FAQ

페이지 범위는 어떻게 입력하나요?

1-3,5,7-9처럼 입력하면 됩니다. 이 예시는 1페이지부터 3페이지, 5페이지, 7페이지부터 9페이지를 추출합니다.

페이지 번호는 0부터 시작하나요?

아니요. 사용자가 입력하는 페이지 번호는 일반적인 문서처럼 1부터 시작합니다.

PDF 병합 관련 FAQ

PDF 병합 순서는 어떻게 정해지나요?

파일 선택창에서 선택된 순서대로 병합됩니다. 원하는 순서가 중요하다면 파일 이름을 정렬하기 쉽게 바꾼 뒤 선택하는 것을 권장합니다.

여러 개의 큰 PDF도 병합할 수 있나요?

가능하지만 모든 작업이 브라우저에서 이루어지므로 기기 메모리와 브라우저 성능에 따라 처리 시간이 길어질 수 있습니다.

파일 보안 관련 FAQ

선택한 PDF가 서버에 업로드되나요?

아니요. PDF 파일은 서버로 전송되지 않고 사용자의 브라우저 내부에서 처리됩니다.

민감한 문서를 사용해도 되나요?

이 사이트는 파일을 업로드하지 않는 방식으로 설계되어 있지만, 공용 컴퓨터에서는 작업 후 다운로드 파일과 브라우저 기록을 직접 확인하는 것이 좋습니다.
